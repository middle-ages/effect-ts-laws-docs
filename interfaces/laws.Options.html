Interface Options<Class, F, A, B, C, R, O, E>

Options for testing parameterized-type typeclasses. All the typeclass laws here expect their arguments to be of this type.

interface Options<Class, F, A, B, C, R, O, E> {
    F: Kind<Class, R, O, E, F>;
    a: Arbitrary<A>;
    b: Arbitrary<B>;
    c: Arbitrary<C>;
    equalsA: Equivalence<A>;
    equalsB: Equivalence<B>;
    equalsC: Equivalence<C>;
    getArbitrary: LiftArbitrary<F, R, O, E>;
    getEquivalence: LiftEquivalence<F, R, O, E>;
}

a: Arbitrary<A>

An arbitrary for the underlying type. For example, when testing Option instances, and the type parameter A is number, this type would be set at fc.Arbitrary<number>. The monomorphic runner set this type, and the other two underlying type arbitraries, to readonly number[].

getArbitrary: LiftArbitrary<F, R, O, E>

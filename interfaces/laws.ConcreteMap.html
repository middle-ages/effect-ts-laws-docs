Interface ConcreteMap<A>

Type-level map of typeclass name to type lambda and law list type for typeclasses on concrete types on the underlying type A. Used to map from typeclass name to its various test type. For example, to get the types related to the Monoid laws for the Option instance on number:

Example

type MyMonoidTypes = ConcreteMap<number>['Monoid]
// MyMonoidTypes ≡ {
//   lambda: MonoidTypeLambda
//   laws: LawSet<[[a: number], [a: number]]>
// }

Use module augmentation to add entries here for a new typeclasses.

interface ConcreteMap<A> {
    Equivalence: {
        lambda: EquivalenceTypeLambda;
        laws: LawSet<[[A, A, A], [A, A], [A]], []>;
    };
    Monoid: {
        lambda: MonoidTypeLambda;
        laws: LawSet<[[A], [A]], [LawSet<[[A, A, A], [A, A, A]], []>]>;
    };
    Order: {
        lambda: OrderTypeLambda;
        laws: LawSet<[[A, A, A], [A, A], [A], [A, A]], []>;
    };
    Semigroup: {
        lambda: SemigroupTypeLambda;
        laws: LawSet<[[A, A, A], [A, A, A]], []>;
    };
}

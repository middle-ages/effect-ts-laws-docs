Interface ApplicativeOptions<F, A, B, C, In1, Out2, Out1>

Options for testing parameterized-type typeclasses. This is the intersection of all the option types required for all laws. Every typeclass laws test requires these options, besides any custom requirements.

The type lambda Class is for the typeclass under test and F is for the parameterized data type under test.

interface ApplicativeOptions<F, A, B, C, In1, Out2, Out1> {
    F: Applicative<F & TypeLambda>;
    a: Arbitrary<A>;
    b: Arbitrary<B>;
    c: Arbitrary<C>;
    equalsA: Equivalence<A>;
    equalsB: Equivalence<B>;
    equalsC: Equivalence<C>;
    getArbitrary: LiftArbitrary<F, In1, Out2, Out1>;
    getEquivalence: LiftEquivalence<F, In1, Out2, Out1>;
}
